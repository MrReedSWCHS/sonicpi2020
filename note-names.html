3 Note Names

Experiment using these code improvements for 5 mins

3.1 Options and release:

Use amp: to change the volume (amp: 1 is default)

play :cs5, amp: 1.5

Use release to change the release portion of the synth envelope (how long the note lasts).

play :cs5, release: 0.2

3.2 Note names

play 61 is equal to play :cs4

Sharp = s
Flat = f

E.g. :cs5 and :ef3
